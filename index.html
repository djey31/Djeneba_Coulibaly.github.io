Automated Classification of Dry Bean Varieties — Supervised Learning

Janvier 2026 — Projet académique (AMSE)

Projet de classification supervisée visant à automatiser l’identification de variétés de haricots secs afin de réduire l’asymétrie d’information liée au tri manuel. :contentReference[oaicite:1]{index=1}

  • Données : UCI Dry Bean Dataset — 13 543 observations, 7 variétés, 16 descripteurs morphologiques (area, perimeter, shape factors…).
  • Pipeline : EDA → preprocessing (RobustScaler) → gestion du déséquilibre (SMOTE) → comparaison modèles + validation cross-validation stratifiée (5-fold).
  • Modèles testés : Logit/Logistic Regression, Elastic Net, SVM (RBF), MLP.
  • Résultat clé : le meilleur modèle est SVM (RBF) en espace de features original (sans PCA), avec Macro-F1 = 0.9369 et Balanced Accuracy = 0.9357.
  • Insight : la réduction de dimension (PCA) peut dégrader la séparabilité malgré une variance expliquée élevée.

Stack : Python, scikit-learn, PCA, SMOTE, SVM, MLP, EDA
